# Blockchain-based Secure Communication in Wireless Sensor Networks

## Overview

This project explores the integration of blockchain technology into Wireless Sensor Networks (WSNs) to improve data security, transmission efficiency, and reliability.

The system introduces a blockchain-based architecture that optimizes communication between sensor nodes while ensuring data integrity and filtering irrelevant data.

---

## Problem Statement

Traditional WSNs face challenges such as:

- inefficient data transmission paths  
- security vulnerabilities  
- unreliable data delivery  
- excessive noise in transmitted data  

This project addresses these issues using blockchain-based validation and routing optimization.

---

## Solution

A blockchain-integrated architecture was designed where:

- sensor nodes communicate through optimized routes  
- blockchain acts as a decentralized validation layer  
- only verified and relevant data is transmitted  
- data integrity is ensured through distributed consensus  

---

## Key Concepts Used

- Blockchain integration in WSN  
- RAFT consensus algorithm  
- Data filtering and validation  
- Distributed communication systems  
- Network simulation using Omnet++  

---

## Architecture

The system includes:

- Base station  
- Blockchain servers  
- Sensor nodes (clients)  
- Consensus mechanism (RAFT)  

The blockchain layer ensures secure and validated communication between nodes.

---

## Implementation

- Simulated network using Omnet++  
- Modeled communication between sensor nodes and blockchain layer  
- Implemented message routing and node interaction  
- Tested multiple scenarios including:
  - node communication
  - server failure
  - data transmission flows  


---

## Results

The system demonstrated:

- improved data transmission efficiency  
- better node communication  
- enhanced data security  
- reduction of irrelevant data transmission  

---

## Research Contribution

This project proposes a novel approach where blockchain is used as:

- a routing optimization layer  
- a validation and filtering system  
- a mechanism to improve trust in WSN communication  

---

## Limitations

- simulation-based (not deployed in real hardware)  
- scalability challenges in large WSNs  
- energy efficiency considerations
